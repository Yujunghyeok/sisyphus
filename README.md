## 🚀 Getting Started

sisyphus shows a real-time ASCII scene of Sisyphus pushing a boulder up a slope. When the boulder reaches the top, it rolls back down. Sisyphus follows, and the loop starts again.

Use it if you want a small terminal app that runs on Windows and plays a calm, looping ASCII animation.
